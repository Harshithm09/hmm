Hidden Markov Model using Baum Welch Algorithm


Name : Harshith M

Description

This project implements a Hidden Markov Model using the Baum Welch Algorithm.

The program learns the parameters of the model using observation sequences and displays a state transition diagram.


Files

baumwelch.py
Implementation of Baum Welch Algorithm

visualize.py
Draws the state transition diagram

main.py
Main program


Run

Install libraries

pip install -r requirements.txt


Run program

python main.py
